This project is a simple web scraping exercise where I collected product listings from Jiji using Python. 
The goal was to practice extracting real data from a website and turning it into a structured dataset that can be analyzed later.
Using Python and BeautifulSoup, I scraped product information from the listings page and organized the data into a Pandas DataFrame.
Tools Used
	•	Python
	•	BeautifulSoup
	•	Requests
	•	Pandas
